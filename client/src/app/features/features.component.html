Features

Responsive Design Media Queries and CSS

Use [ngClass] combined with media queries built into Angular's BreakpointObserver to determine the appropriate CSS class for the device.

Material Components Angular Material Design

Commonly used Material design modules are imported and ready to use along with a prebuilt Material theme to help with your UI/UX and color scheme.

Navigation Boilerplate Toolbar and Hamburger Mobile Nav

Easy to build upon Navigation setup for mobile and desktop.

Basic NodeJS Server Backend Setup with Mock Database

Basic NodeJS, Express, and Json Server setup for middle-tier and backend. Develop a simple website or extend the template for a full-scale MEAN stack web application.
